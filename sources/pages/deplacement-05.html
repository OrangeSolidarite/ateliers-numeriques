S'entraîner avec un menu déroulant

Objectif : Ouvrir le menu déroulant et sélectionner le chiffre 3

💡 Conseil important

Cliquez sur la petite flèche du menu pour ouvrir la liste. Faites défiler jusqu'à voir le chiffre 3, puis cliquez précisément dessus.

Comment utiliser un menu déroulant

Étape 1 - Ouvrir le menu :
Cliquez sur la flèche à droite du champ pour afficher la liste des choix.
Étape 2 - Faire défiler :
Descendez dans la liste jusqu'à ce que le chiffre 3 apparaisse.
Étape 3 - Sélectionner :
Cliquez bien sur le chiffre 3.
Astuce :
Prenez votre temps. Le bon geste vient avec l'entraînement.
