Envoyer

1 - Cliquer dans la zone A:
puis taper l'adresse mail du destinataire
par exemple a.dreuilhe@cyber-base.org

Attention en minuscule pas d'accent ni espace

2 - Cliquer dans la zone Objet :
puis taper l'objet du message
par exemple invitation expo d'Auvillar dimanche

3 - Cliquer dans la zone blanche du bas
puis taper le contenu du message
4 -Puis cliquer sur le bouton envoyer
(en haut à droite)

Vous allez apprendre à envoyer un message électronique, un mail, un courriel

Pour envoyer un message il faut 3 informations

  • l'adresse mail de la personne du type xxxxx@xxxxx.fr à taper sans espace, sans majuscule, sans accent
  • L'objet du message
  • Le texte du message
