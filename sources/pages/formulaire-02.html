Formulaire 2

Effacer du texte

Objectif : Apprendre à effacer du texte avec la touche Retour arrière, située à droite de votre clavier.

💡 Conseil important

La touche Retour arrière efface le caractère placé juste avant le curseur clignotant.

Instructions pas à pas

Étape 1 - Cliquer dans le champ Prénom :
Le trait vertical clignotant apparaît dans la zone : vous êtes prêt à taper.
Étape 2 - Taper votre prénom :
Saisissez votre prénom normalement dans le champ.
Étape 3 - Effacer lettre par lettre :
Appuyez sur le retour arrière ← pour effacer la dernière lettre. Chaque appui efface un caractère de plus.
Étape 4 - Recommencer :
Une fois le champ vide, tapez un autre prénom pour vous entraîner à nouveau.
Étape 5 - Effacer tout d'un coup :
Le bouton Effacer remet le champ à zéro en un seul clic.

À vous de jouer !

Tapez votre prénom, puis effacez-le avec la touche Retour arrière :

Exemple

Si vous avez tapé « Marie », chaque appui sur le retour arrière efface une lettre :

Marie
→
Mari
→
Mar

Suite

Rappel des touches utiles

Retour arrière ← : Efface le caractère situé juste avant le curseur
Suppression suppr : Efface le caractère situé juste après le curseur
Tab ↹ : Passer d'un champ au suivant sans utiliser la souris
