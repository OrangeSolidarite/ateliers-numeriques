Copier-Coller

Apprendre le Copier / Coller

Objectif : Copier les réponses de la colonne de droite et les coller dans les questions correspondantes à gauche

💡 Comment faire ?

Le copier-coller permet de dupliquer du texte d'un endroit à un autre sans avoir à le retaper.

1️⃣ Copier les réponses

Sélectionner le texte :
Dans la colonne de droite, cliquez sur une réponse :
  • 2 clics pour sélectionner 1 mot
  • 3 clics pour sélectionner 1 ligne
Copier :
Faites un clic droit sur le mot sélectionné, puis cliquez sur Copier dans le menu qui apparaît.

2️⃣ Coller les réponses

Positionner le curseur :
Cliquez dans la case blanche à côté de la question (colonne de gauche). Vérifiez que le curseur clignotant apparaît bien dans la case.
Coller :
Faites un clic droit dans la case, puis cliquez sur Coller dans le menu qui apparaît.
