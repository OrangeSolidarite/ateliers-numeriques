Menu déroulant

Apprendre à utiliser un menu déroulant

Objectif : Sélectionner le chiffre 3 dans le menu déroulant

💡 Qu'est-ce qu'un menu déroulant ?

Un menu déroulant permet de choisir une option parmi plusieurs. Il économise de l'espace en affichant les choix uniquement quand on clique dessus.

Menu déroulant

👆 Cliquez sur la petite flèche noire pour ouvrir la liste de choix

Instructions

1. Ouvrir le menu :
Cliquez sur la petite flèche noire à droite du menu. Une liste de nombres va s'afficher.
2. Parcourir la liste :
Déplacez votre souris dans la liste déroulante jusqu'à ce que le chiffre 3 apparaisse en blanc sur fond bleu.
3. Sélectionner :
Cliquez bien sur le chiffre 3, pas à côté. Soyez précis !
4. Validation :
Si vous avez cliqué sur le bon chiffre, vous passerez automatiquement à la page suivante.

Astuces

⚠️ Attention : Si vous cliquez sur un autre nombre, vous serez redirigé vers une page d'erreur
💡 Conseil : Prenez votre temps pour bien viser le chiffre 3
🔄 Recommencer : Si vous vous trompez, utilisez le bouton "Retour" de votre navigateur
