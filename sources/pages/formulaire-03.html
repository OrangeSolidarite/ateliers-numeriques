Formulaire 3

Remplir un formulaire complet

Objectif : Remplir tous les champs du formulaire comme on le ferait sur un vrai site internet

💡 Conseil important

Lisez bien chaque champ avant de taper. Certains champs demandent un format précis, comme la date de naissance. Prenez votre temps et vérifiez ce que vous avez écrit.

Instructions pas à pas

Étape 1 - Civilité :
Cliquez sur le petit cercle à côté de Monsieur, Madame ou Mademoiselle selon votre situation.
Étape 2 - Prénom et Nom :
Cliquez dans le champ Prénom, tapez votre prénom, puis faites de même pour le Nom.
Étape 3 - Adresse :
Remplissez l'adresse, le code postal et la ville dans les champs correspondants.
Étape 4 - Date de naissance :
Tapez votre date au format jj/mm/aaaa. Exemple : 14/08/1959
Étape 5 - Code à recopier :
Recopiez exactement le code affiché en bleu, en respectant les majuscules et les minuscules.
Étape 6 - Listes :
Sélectionnez votre département dans la liste, puis votre région dans le menu déroulant.
